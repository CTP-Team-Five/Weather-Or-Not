# ADR 0004: Use Open-Meteo

## Status
Accepted

## Context
Needed an API to call for weather information.

## Decision
We will use **Open-Meteo** for this project.

## Benefits
- Free access for non-commercial use.
- No API key needed.
- Up to 16 days of information.
- Gives detailed and tailored weather information with each call with the feature to  filter what's wanted in specifics.

## Consequences
- Searches *only* with Longitude and Latitude
